Un constructor tiene la responsabilidad de poner cada objeto de esa clase en un estado adecuado para que pueda ser usado una vez que haya sido creado.

  • Esta operación se denomina inicialización.

Uno de los rasgos distintivos de los constructores es que tienen el mismo nombre que la clase en la que son definidos. 


public class MaquinaDeBoletos { 
    Se omitieron los campos 
    /** 
    * El Código muestra el constructor de la clase MaquinaDeBoletos.
    */ 
    public MaquinaDeBoletos (int precioDelBoleto) { 
        precio = precioDelBoleto; 
        saldo = 0; 
        total = 0; 
    } 
    Se omitieron los métodos 
}

En Java se requiere llamar al constructor para cada objeto creado. Donde al momento de escribir new se esta llamando al constructor de la clase para realizar la inicialización.

La sintaxis para llamarlo es: 


TipoClase variable = new TipoClase(argumentos);
/** 
* Ejemplo :
*/ 
Asignatura asignatura1 = new Asignatura(int creditos);

Al igual que los métodos, los constructores también pueden recibir parámetros como se observa en el ejemplo el constructor recibe un valor entero precioDelBoleto. 


    public MaquinaDeBoletos (int precioDelBoleto) { }

No obstante, una importante diferencia entre los constructores y los métodos es que los constructores no pueden devolver valores, por lo cual no pueden especiﬁcar un tipo de valor de retorno (ni siquiera void).

Diferencias métodos - constructores


public class LibroCalificaciones
{ 
	// nombre del curso para este LibroCaliﬁcaciones
	private String nombreDelCurso; 
	

	// el constructor inicializa nombreDelCurso con el objeto String que se provee como argumento
	public LibroCaliﬁcaciones( String nombre ) 
	{ 
		// inicializa nombreDelCurso
		nombreDelCurso = nombre; 
	} // ﬁn del constructor

	// método para establecer el nombre del curso
	public void establecerNombreDelCurso( String nombre )
	{ 
		// almacena el nombre del curso
		nombreDelCurso = nombre; 
	} // ﬁn del método establecerNombreDelCurso
}

Cabe mencionar que el compilador de Java de manera predeterminada, proporciona un constructor predeterminado el mismo que carece de parámetros, en cualquier clase que no incluya un constructor en forma explícita.
