Conceptos Básicos

Antes de aprender cómo realizar la creación de un diagrama de clase en DIA debemos conocer que se entiende por:

  • Diagrama de Clases
  • UML

Diagrama de Clases

El diagrama de clases de UML se usa para describir los tipos de objetos en un sistema y las relaciones entre estos.

Además muestra las propiedades y las operaciones de las clases y qué restricciones hay para las relaciones entre los objetos.

UML

Unified Modeling Language (UML) es un lenguaje gráfico usado para visualizar, especificar, y documentar sistemas informáticos mediante la construcción de diagramas.


Cabe recalcar que para comenzar el desarrollo de un proyecto es necesario el análisis del mismo y la construcción de su estructura para evitar futuros problemas y lograr conseguir una buena practica de desarrollo.

Paso 1

Se selecciona las herramientas para diagramas UML en la barra lateral mostrando en la parte superior herramientas generales seguidas a continuación de las herramientas que nos permitirán crear nuestro diagrama UML.

Paso 2

El primer elemento de las herramientas UML nos permite crear una clase para lo cual se lo selecciona y se marca algún punto del área de trabajo donde se ubicara la clase.

Paso 3

Se nos muestra una clase simple sin atributos ni métodos en el lugar marcado por lo que para modificar dicha clase se realiza doble clic para acceder a las propiedades de las misma.

Paso 4

Con ayuda de cada una de las herramientas que encontramos en la barra lateral nos permitirán crear las relaciones entre las diferentes clases:

  • Asociación (conexión entre clases)
  • Dependencia (relación de uso)
  • Generalización/especialización (relaciones de herencia)
